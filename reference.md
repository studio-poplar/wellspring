# 参考文献・科学的知識ベース

## 基本知識

### 肌の構造と機能
- **肌のpH値**：健康な肌は弱酸性（pH 4.5-5.5）で、雑菌の増殖を抑える
  - 参考：日本化粧品工業会「化粧品の基礎知識」
- **皮脂の役割**：肌を保護し水分蒸発を防ぐ（バリア機能）
  - 参考：日本皮膚科学会「スキンケアのすすめ」
- **角質層**：肌の最外層で、水分保持とバリア機能を担う

### 乾燥とニキビの関係
- **乾燥肌のメカニズム**：角質層の水分不足 → バリア機能低下 → 皮脂過剰分泌
- **皮脂と毛穴詰まり**：過剰な皮脂 + 古い角質 → 毛穴詰まり → アクネ菌増殖 → ニキビ
  - 参考：American Academy of Dermatology「Acne: Overview」

### スキンケアの基本3ステップ
1. **洗顔**：弱酸性の洗顔料で朝夜1回ずつ（肌のpHを保つ）
2. **保湿**：化粧水 → 美容液 → クリームで水分と油分のバランス
3. **日焼け対策**：紫外線ダメージによる乾燥・シミ予防

## 栄養・サプリメント

### コラーゲン
- **役割**：肌の弾力性を保つタンパク質、加齢で減少（20代をピークに低下）
- **補給方法**：食事（豚軟骨など）またはサプリメント
  - 参考：「Journal of Cosmetic Dermatology」（2019）

### ビタミンC
- **役割**：抗酸化作用、コラーゲン生成促進、シミ予防
- **補給方法**：オレンジなどの柑橘類、またはサプリメント

### ビタミンE
- **役割**：抗酸化作用、肌老化予防、紫外線ダメージ軽減
- **補給方法**：アーモンド、ひまわり油、またはサプリメント
